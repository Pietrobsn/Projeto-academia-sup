# 🏋️ Projeto Academia Superação

Sistema administrativo desenvolvido para auxiliar na organização e no controle interno de uma academia, reunindo ferramentas essenciais para gestão financeira, folha de pagamento e solicitações de cancelamento/reembolso.

O projeto foi pensado para facilitar processos do dia a dia, reduzir erros manuais, melhorar a visualização dos dados e deixar a administração mais prática, rápida e profissional.

---

## 📌 Módulos do Sistema

O projeto é composto por três sistemas principais:

### 💰 Sistema de Caixa

Controle de entradas financeiras da academia, permitindo registrar e acompanhar movimentações por unidade, forma de pagamento e período.

**Principais funcionalidades:**

- Cadastro de lançamentos financeiros
- Controle por unidade
- Registro por forma de pagamento:
  - Pix
  - Dinheiro
  - Débito
  - Crédito
- Histórico de lançamentos
- Filtros para consulta
- Impressão de relatórios
- Exportação de dados
- Painel com totais e indicadores

---

### 📄 Sistema de Folha de Pagamento

Sistema criado para organizar os pagamentos de colaboradores, professores, estagiários e funcionários da academia.

**Principais funcionalidades:**

- Cadastro de funcionários
- Separação por categoria
- Controle de salário, vale, descontos e adicionais
- Cálculo de valores a receber
- Controle de status de pagamento
- Resumo geral da folha
- Visualização em painel administrativo
- Impressão de relatórios
- Organização por unidade

---

### ❌ Sistema de Cancelamento e Reembolso

Ferramenta para registrar, acompanhar e organizar solicitações de cancelamento de planos e possíveis reembolsos.

**Principais funcionalidades:**

- Cadastro de solicitação de cancelamento
- Registro de dados do aluno
- Controle por unidade
- Informações do plano
- Motivo do cancelamento
- Dados para reembolso
- Status da solicitação
- Área administrativa para acompanhamento
- Organização dos pedidos enviados

## 🎯 Objetivo do Projeto

O objetivo deste projeto é centralizar ferramentas importantes para a rotina administrativa de academias, oferecendo uma interface prática, visual e funcional para facilitar o controle interno.

A proposta é transformar processos que normalmente são feitos em planilhas ou anotações manuais em sistemas mais organizados, com melhor visualização e maior segurança na gestão das informações.

---

## ✅ Benefícios

- Melhor organização dos dados
- Redução de erros manuais
- Facilidade na consulta de informações
- Interface visual e moderna
- Controle por unidade
- Relatórios para impressão
- Possibilidade de integração com banco de dados
- Mais agilidade na rotina administrativa
